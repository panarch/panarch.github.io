I just started my own blog!
It's a first article and so let's begin by writing whatever.

Let me simply introduce what the tool I used for building this blog.
And my choice is Zola!

Zola is A fast static site generator in a single binary with everything built-in.

Before I decided what blogging platform to use, my requirements were these below:

  1. Markdown support

It's very natural to use WYSIWYG editors to write article, yeah, but...
At this point, I wanted to manage my blog contents using git but using WYSIWIG editors it can be quite tough.
In usual cases, contents generated by WYSIWIG editors are not only that human-readable, but also it is hard to replace an existing WYSIWIG editor to other one.

Getting stuck at something, that's something I'd really liked to avoid.

Then Markdown can be a very proper choice.
It's very portable and Markdown is also human-readable itself.
I don't need to worry about this kind of scenario: what should I do when my blogging platform is shutdown or outdated..?
Using Markdown, it's quite easy to move to any other blogging platforms.

  1. Static site generator

Nowadays Web is not only for making something documents to read.
We can now make almost whatever in web development environment.

And for me, that was even the main source of my income for a last few years.
I made lots of complex websites and web based applications.
But for blogging, that can be too much.
Static HTML is just enough for building something... website for readings.

...blah blah blah...

I... just don't want to take care about updating React version for running my blog.

  1. Written in Rust

Very simple reason, I like Rust, I love Rust so if there exists static site generator written in Rust, why not using that?
Maybe I can even contribute to the project in the future.

If that static site generator is written in PHP, though I choose that, I will not ever consider to contribute to the project.
But for Rust project, I can happily do something for free.

Zola

And Zola satisfies all 3 requirements, so I chose it.
